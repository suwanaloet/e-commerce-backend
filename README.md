# E-commerce Back End 

### Description
backend for e-commerce website made in Express, with MySQL using Sequelize. 

### Installation

#### Command line start:

##### mysql -u root -p
##### enter password
##### source Develop/db/schema.sql
##### show databases

#### In Terminal

##### npm run seed
##### npm run start


### Necessary Tools
##### npm: npm init
##### express: npm i express
##### sqlite: npm i sqlite
##### mysql2: npm i mysql2
##### dotenv: npm i dotenv
##### sequelize: npm i sequelize
